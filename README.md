# **Netflix Comments Browser Extension**

##### *A Safari and Chrome Extension which injects an option on the Netflix’s web player to add comments.*
&nbsp;

----
## Tech

* Java Script
* Swift
* Amazon’s DynamoDB
* Amazon’s Lambda Function
* Amazon’s API Gateway
* HTML
* CSS
